## 배포 방법 (Vercel — 무료)

### 1단계: GitHub에 올리기

1. https://github.com 에서 무료 계정 만들기
2. 새 저장소(Repository) 만들기 → 이름: `dream-journey`
3. 이 폴더의 파일들을 모두 업로드

### 2단계: Vercel에 배포하기

1. https://vercel.com 에서 GitHub 계정으로 로그인
2. "New Project" → GitHub 저장소 선택
3. 설정 그대로 두고 "Deploy" 클릭
4. 완료! `dream-journey-xxx.vercel.app` 주소가 생성됩니다

### 3단계: 도메인 연결 (선택)

- Vercel 대시보드 → Settings → Domains
- 구입한 도메인 입력 (ex: dreamjourney.kr)

---

## 로컬에서 실행하기

```bash
npm install
npm run dev
```
